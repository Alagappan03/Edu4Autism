## 📖 About The Project

Edu4Autism is an interactive mixed reality platform designed to make developmental therapy more engaging, accessible, and comfortable for autistic children.

Traditional therapy methods can sometimes feel repetitive or difficult to sustain for long periods. Edu4Autism transforms therapy-oriented activities into playful and interactive experiences using realtime pose tracking, gamification, and camera-based interaction.

The platform focuses on:

- Motor skill development
- Coordination improvement
- Cognitive engagement
- Behavioral growth
- Interactive participation

while ensuring that the experience remains child-friendly and avoids sensory overload or excessive virtual dependency.

---

## ✨ Core Features

### 🎮 Gamified Therapy Interaction

Transforms developmental exercises into playful mixed reality experiences to improve engagement and participation.

### 📷 Realtime Pose Detection

Uses Google MediaPipe and camera-based tracking to monitor movement and interaction in realtime.

### 🧠 Development-Focused Activities

Interactive exercises are designed to support:

- Motor skills
- Coordination
- Eye contact
- Cognitive engagement

### 🧩 Child-Friendly UI/UX

Designed with simplified layouts, responsive interfaces, smooth interaction flow, and accessibility-focused visual design.

### 📊 Performance Tracking

Tracks participation and progress to support developmental monitoring and engagement feedback.

### 💻 Cross-Device Accessibility

Works across commonly available devices without requiring expensive therapy hardware.

## 🎨 Design Philosophy

Edu4Autism was designed with one core principle:

> Therapy should feel supportive and engaging, not overwhelming.

Every interaction, animation, color choice, and gameplay flow was designed to:

- Reduce sensory overload
- Maintain emotional comfort
- Encourage active participation
- Create positive engagement without excessive screen dependency

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Unity Hub and Unity Editor (Check `ProjectSettings/ProjectVersion.txt` for the exact version).
* A webcam (for testing in the editor) or a mobile device (Android/iOS) for deployment.

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/Alagappan03/Edu4Autism.git
```

2. **Open in Unity**
   * Open Unity Hub.
   * Click `Add` and select the cloned `Edu4Autism` folder.
   * Click on the project name to open it in the Unity Editor.

3. **Run the Scene**
   * Navigate to the `Assets/Scenes` folder in the Project window.
   * Double-click the main scene (often named `SampleScene` or `MainMenu`) to load it.
   * Press the Play button at the top of the editor.
---
## 📱 How to Use

1. Ensure your device or computer has a working camera.
2. Launch the application.
3. Allow camera permissions when prompted.
4. Point the camera at the specific markers or interact with the screen as directed by the on-screen tutorial.
